# Supabase connection

This project is a **Vite + React + TypeScript** app and is already wired to Supabase using the official JS client.

## Configure

- Copy `.env.example` to `.env`.
- Fill in `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY` from Supabase Dashboard → Project Settings → API.

## Run

```bash
npm i
npm run dev
```

## Important

Only use the anon key in the frontend.
